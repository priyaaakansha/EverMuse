Cancellation

If the order has already dispatched from our warehouse:Your cancellation request will be processed and refund will be initiated for your order. Incase the order is still "Out for delivery" and you are contacted by our courier partner kindly refuse the accept it.


If the order has already been delivered: We request you to please raise a return request for the order. Please for through the Return FAQ's for the same.


If the order is a Reserve: If the order is "Ready to collect" at the store we request you to kindly get in touch with the store with your order ID and they shall initiate the cancellation request for your order.
